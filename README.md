# webpack4多入口配置

基于 webpack4 的多入口配置，支持 Vue + html 渐进式写法，支持 element 等第三方框架的引用。

```bash
# clone the project
https://github.com/ChaterV/webpack4-multi-entry.git

# enter the project directory
cd webpack4-multi-entry

# install dependency
npm install

# develop
npm run dev

# production
npm run build
```
